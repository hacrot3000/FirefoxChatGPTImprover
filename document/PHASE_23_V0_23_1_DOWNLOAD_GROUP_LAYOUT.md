# Phase 23 v0.23.1 — Dedicated Managed download group

The managed-download controls are now in their own collapsible sidebar group directly below **New target element**. The move is presentational only: field IDs, local-action profile storage, per-tab overrides, URL routing, immutable execution snapshots, recovery, and retry behavior are unchanged.

The Local action profile group remains the place to assign, route, save, or reset the profile/tab override that owns both download and shell settings.
